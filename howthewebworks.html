How The Web Works

Part One: Solidify Terminology

1. What is HTTP?

A: HTTP stands for Hyper Text Transfer Protocol that governs the data that transfers between client and servers.

2. What is URL?

A: URL stands for Uniform Resource Locator which is an address for internet resource.

3. What is DNS?

A: DNS stands for Domain Name System that takes human-readable URLs and converts them into IP addresses.

4. What is a query string?

A: A query string is a part of the URL that assigns value to the specified parameters.

5. List two HTTP Verbs and their use cases.

A: 1. Get - gets data from the servers 2. Post - changes data on servers

6. What is an HTTP request?

A: It is an URL request from a client to the server.

7. What is an HTTP response?

A: It is a response sent by the server to the client as per their request.

8. What is an HTTP header? Give a couple examples of request and response headers you have seen.

A: HTTP header provide additional information about the request or response . For example Request Headers: Host, User-Agent, Accept, Cookie, Cache-Control Response Headers: Content-Type, Last-Modified, Set-Cookie, Cache-Control

9. What happens when you type a “http://somesite.com/some/page.html” in a browser?

A: 1. The browser uses DNS to find the IP address for this site. 2. The request is made to the IP address. 3. The server sends a response. 4. The browser makes a DOM from that response.
